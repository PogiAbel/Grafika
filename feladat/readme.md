## Féléves feladat

Tűz animáció, hangulatos közegben.

Interakciók:

 - Térben mozogni
 - Tűz részecskék manipulálása: rászecskeszám, -méret, -élettartam, -kilengés
 - Tűz fényének módosítása: piros, zöld, kék
 - Help menu
